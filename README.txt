Kinectifier Quartz Composer Plugin for Resolume Avenue
Copyright 2011 Tyler Freeman
http://odbol.com

A Quartz Composer patch for using 3d video feeds from the Xbox Kinect in Resolume Avenue, 
with effects and textures triggerable from Resolume.

Used in the VJacket demo installation at Maker Faire Bay Area 2011. ( http://vjacket.com )

Based on the KinectTools and OpenKinectBeta plugins for Quartz Composer 
( http://kineme.net/forum/Discussion/Programming/v002OpenKinectBeta ) and many patches/ideas 
from George Toledo ( http://www.georgetoledo.com/ ) and the Kineme community ( http://kineme.net/ ) 

INSTALLATION:

1. Download the KinectTools plugin for QC: http://kineme.net/KinectTools
2. Copy KinectTools.plugin to "/Library/Graphics/Quartz Composer Patches/"
3. If it doesn't work, you may need other plugins as well, such as: 

AlphaBlendMode.plugin
GLTools.plugin
StructureTools.plugin
v002 Dilate.plugin
v002 Erode.plugin
v002Glitch.plugin


USAGE:

1. Connect the Kinect to your computer.
2. Drop the .qtz file into Resolume or drag it from the file browser in Resolume.
